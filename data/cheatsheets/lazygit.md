# lazygit

TUI for git.

## Launch
```sh
lazygit                          # open in current repo
lazygit -p /path/to/repo        # specific repo
```

## Navigation
```
h/l                              # switch panels
j/k                              # move up/down
enter                            # expand/view
q                                # quit
?                                # help / all keybindings
```

## Staging & Committing
```
space                            # stage/unstage file
a                                # stage all
c                                # commit
C                                # commit with editor
A                                # amend last commit
```

## Branches
```
n                                # new branch
space                            # checkout branch
M                                # merge into current
r                                # rebase onto current
```

## Stash
```
s                                # stash
g                                # pop stash
```

## Other
```
p                                # pull
P                                # push
f                                # fetch
d                                # view diff
/                                # filter/search
```
